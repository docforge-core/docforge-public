# DocForge

Generate professional documents, PDFs and templates directly from structured data.

Website:
https://docforge.website

## Features

- Salary slip generator
- Invoice generation
- PDF/A-3 export
- GST-ready templates
- Payroll documents
- Business document generation
- AI-assisted document workflows

## Use Cases

- HR teams
- Freelancers
- Small businesses
- Accounting workflows
- Payroll processing
